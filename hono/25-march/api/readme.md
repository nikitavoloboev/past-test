Based of [Hono example](https://github.com/honojs/examples/tree/main/blog).

## Setup

```
bun i
```

## Run

```
bun dev
```

## Getting it working with [Turso](https://turso.tech)

Follow [this](https://docs.turso.tech/sdk/ts/guides/hono) for setup.

Need this `.env`:

```
TURSO_DATABASE_URL=
TURSO_AUTH_TOKEN=
```
